# js-twitter-clone
Full-Stack JavaScript Twitter clone with MongoDB, Express.js, React.js, and Node.js

```
const userSchema = new Schema({
    handle: {
        type: String,
        required: true
    },
    email: {
        type: String,
        required: true
    },
    password: {
        type: String,
        required: true
    },
    date: {
        type: Date,
        default: Date.now
    }
});
```
